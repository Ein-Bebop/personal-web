Diego Acosta is a software engineer building his own personal portfolio/website from scratch with a custom Node.js static site generator (no frameworks). He values:

- Minimalist, content-first design — inspired by karpathy.bearblog.dev
- System fonts, centered single-column layout, ~700px max-width
- Light/dark mode via CSS media query only — no JavaScript on the live site
- New posts added by dropping .md files into a posts/ folder
- Clean semantic HTML and modern CSS — no heavy abstractions
- Two main sections: Home (bio/work history) and Blog (date + title only, no summaries or tags)
